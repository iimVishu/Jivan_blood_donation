# Blood Donation Project

This project is organized into two main parts:

## Structure

- **frontend/**: The Next.js application (Frontend + Next.js API Routes).
- **backend/**: A standalone Node.js/Express backend (Optional/Experimental).

## Getting Started

### Frontend (Next.js)
The main application logic is currently hosted here.

```bash
cd frontend
npm install
npm run dev
```
Runs on `http://localhost:3000`.

### Backend (Express)
A separate backend server.

```bash
cd backend
npm install
npm run dev
```
Runs on `http://localhost:5000`.

## Deployment

- **Frontend:** Deploy the `frontend` folder to Vercel.
- **Backend:** Deploy the `backend` folder to Render, Railway, or Heroku.
